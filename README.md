# Car Rental Platform

A modern, full-stack car rental web application that connects car owners with renters. Built with React and Firebase, this platform offers a seamless experience for browsing, listing, and booking vehicles.

## Features

- **Browse Cars**: Explore a wide selection of available vehicles with detailed specifications, pricing, and high-quality images
- **Secure Authentication**: Sign up and log in using email/password or Google authentication powered by Firebase
- **List Your Car**: Car owners can easily add their vehicles with descriptions, photos, and rental rates
- **Booking System**: Real-time booking management with instant confirmations
- **User Dashboard**: Manage your listings and track your bookings in one place
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices with smooth animations

## Tech Stack

**Frontend**: React, React Router, Tailwind CSS, Framer Motion  
**Backend**: Node.js, Express, MongoDB  
**Authentication**: Firebase Auth  
**Hosting**: Surge (Frontend), Vercel (Backend)
